This repository replicates the results in '[Impact of Foreign Ownership of Bonds on the Mexican Yield Curve](https://drive.google.com/file/d/1cpxxXN2r5Y5-S-2Btk-QikaQkdYNvtCB/view)'.

## Replication Instructions

1.  Clone the repository
2.  Open R in project root
3.  Run: source("run_all.R")

Expected runtime: \~ 30 minutes

If bootstrapped confidence intervals need not be regenerated, the tables and figures in the paper can created in \~3 minutes.
